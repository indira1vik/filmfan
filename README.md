# The FilmFans Website

This project is made with the help of IMDb Api. [IMDb](https://www.imdb.com/).

## Use the website

In the project directory, you can run:
Open [FilmFans Website](http://localhost:3000) to view it in my browser.

**You can also download the zip file try using it.**

This website provides information about the films, television programmes, and shows that are listed in the database and presents the findings in a tidy, simple user interface. The movies can also be added to your Watch List for later viewing. Even after being refreshed, this watch list will be saved. To store the data in cache, we utilised localStorage. You may view the newest and most popular Hollywood and Bollywood films.
